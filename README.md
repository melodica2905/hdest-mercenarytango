## Mercenary Tango
A small addon that randomly spawns a Tango in a map as long as it has deathmatch starts, who when killed, will drop extra goodies.

Has compatibility with Accsensus's Mercenary Merchant addon (A small addon ment to be used with Accensus's Mercenary Merchant addon (https://gitlab.com/accensi/hd-addons/merchant/-/tree/mercenary), where the Tango extra money alongside their other drops.
